# 🏦 Banking Management System (C++)

A simple console-based Banking Management System developed using C++ and Object-Oriented Programming (OOP) concepts.

## 📖 About the Project

This project simulates basic banking operations such as account creation, deposit, withdrawal, and account details display. It is designed for beginners to understand OOP concepts in a real-world scenario.

## ✨ Features

- Create Bank Account
- Deposit Money
- Withdraw Money
- Display Account Details
- Menu-driven interface
- Basic validation checks

## 🧠 Concepts Used

- Classes and Objects
- Encapsulation
- Member Functions
- Conditional Statements
- Loops
- Switch Case
- Input/Output Handling

## 🛠️ Technologies Used

- C++
- OOP (Object-Oriented Programming)
- GCC / G++ Compiler

## ▶️ How to Compile

```bash
g++ banking_system.cpp -o bank
```

## ▶️ How to Run

```bash
./bank
```

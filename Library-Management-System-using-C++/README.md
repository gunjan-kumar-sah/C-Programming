# 📚 Library Management System (C++)

A console-based Library Management System developed using C++ and Object-Oriented Programming (OOP) concepts with File Handling.

---

## 📖 About the Project

This project simulates a real-world library system where users can manage books digitally. It supports adding books, issuing books, returning books, and displaying available books.

Data is stored permanently using file handling.

---

## ✨ Features

- 📘 Add new books
- 📚 Display all books
- 📤 Issue books
- 📥 Return books
- 💾 File handling (data persistence)
- 🔄 Update book status (issued/available)

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes and Objects
- File Handling (ifstream / ofstream)
- Loops and Switch Case
- Conditional Statements
- Data Persistence Logic

---

## 🛠️ Technologies Used

- C++
- File Handling
- GCC / G++ Compiler

---

## ▶️ How to Compile

```bash
g++ library_system.cpp -o library
```

---

## ▶️ How to Run

```bash
./library
```

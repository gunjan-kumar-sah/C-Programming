# 📊 Exam Result Prediction System (C Language + Data Structures)

A console-based **Exam Result Prediction System** built using C language and Data Structures. This project takes student marks as input, analyzes performance, and predicts the final result category like Pass/Fail/Excellent based on logic and scoring patterns.

It is a simple **data structure-based academic analysis system** that helps understand arrays, structures, and basic prediction logic.

---

## 🚀 Features

- 👨‍🎓 Add student details (ID, Name, Marks)
- 📊 Store multiple subject marks
- 🧠 Automatic result prediction system
- 📈 Percentage calculation
- 🏆 Grade classification system
- ⚡ Menu-driven interface

---

## 🧠 Concepts Used

- Structures in C (`struct`)
- Arrays for storing marks
- Functions for modular design
- Loops for processing multiple students
- Conditional statements (if-else)
- Basic data analysis logic

---

## ⚙️ Working Flow

1. Enter student details  
2. Enter marks for subjects  
3. System calculates total and percentage  
4. Predicts result based on performance  
5. Displays final grade and status  

---

## 📊 Result Logic

```

Percentage = (Total Marks / Maximum Marks) × 100

```

### Result Categories:
- 80% – 100% → Excellent 🏆
- 60% – 79% → Good 👍
- 40% – 59% → Average ⚠️
- Below 40% → Fail ❌

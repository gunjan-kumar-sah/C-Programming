# 🚆 Railway Reservation System (Advanced C Project)

A console-based **Railway Reservation System** built using C language with file handling and structured programming concepts.

---

## 📖 About the Project

This project simulates a real-world railway booking system where users can check seat availability, book tickets, cancel tickets, and view all bookings.

It uses file handling to store ticket records permanently.

---

## ✨ Features

- 🔐 Admin Login System
- 💺 View Seat Availability
- 🎟 Ticket Booking System
- ❌ Ticket Cancellation
- 📄 View All Bookings
- 🧾 PNR Generation System
- 💾 File-based Data Storage

---

## 🧠 Concepts Used

- Structures in C
- Arrays
- File Handling (fopen, fprintf, fgetc)
- Conditional Statements
- Loops
- Functions
- String Handling
- Random Number Generation

---

## 🛠️ Technologies Used

- C Language
- GCC Compiler

---

## ▶️ How to Compile

```bash
gcc railway_reservation.c -o railway
```

---

## ▶️ How to Run

```bash
./railway
```

# 🏥 Hospital Management System (Advanced C++ Project)

A fully functional **console-based Hospital Management System** built using C++ with Object-Oriented Programming (OOP) and File Handling concepts.

---

## 📖 About the Project

This project simulates a real hospital system where administrators can manage patient records efficiently. It includes login authentication, patient admission, discharge system, and search functionality.

---

## 🔐 Login System

- Username: `admin`
- Password: `1234`

---

## ✨ Features

- 🔐 Admin Login System
- 🧑 Add Patient Records
- 📋 View All Patients
- 🔍 Search Patient by ID
- 🏥 Discharge Patient System
- 💾 File-based Data Storage
- 🔄 Update Patient Status (Admitted/Discharged)

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation
- File Handling (ifstream / ofstream)
- Authentication System
- Conditional Statements
- Loops and Switch Case
- Real-world system simulation

---

## 🛠️ Technologies Used

- C++
- File Handling
- GCC / G++ Compiler

---

## ▶️ How to Compile

```bash
g++ hospital_system.cpp -o hospital
```

---

## ▶️ How to Run

```bash
./hospital
```

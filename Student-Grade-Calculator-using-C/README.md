# 🎓 Student Grade Calculator (C Language)

A simple C program to calculate a student's total marks, percentage, and grade based on marks obtained in multiple subjects.

## 📖 About

This project takes input marks from the user for multiple subjects, calculates total marks, percentage, and assigns a grade based on performance.

## ✨ Features

- Input multiple subject marks
- Calculate total marks
- Calculate percentage
- Assign grades based on percentage
- Simple and beginner-friendly logic

## 🧠 Logic Used

- Arrays
- Loops
- Conditional Statements (if-else)
- Basic Arithmetic Operations

## 🛠️ Technologies Used

- C Language
- GCC Compiler

## ▶️ How to Run

```bash
gcc grade_calculator.c -o grade
./grade
```

## 📷 Example Output

```text
Enter number of subjects: 5

Enter marks of subject 1: 80  
Enter marks of subject 2: 75  
Enter marks of subject 3: 90  
Enter marks of subject 4: 85  
Enter marks of subject 5: 70  

Total Marks = 400  
Percentage = 80.00%  
Grade = A
```

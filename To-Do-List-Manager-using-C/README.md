# 🧠 Advanced To-Do List Manager (C Language)

A real-world **console-based To-Do List application** built using C programming language with File Handling and Multi-User system.

---

## 📖 About the Project

This project allows multiple users to register, login, and manage their personal tasks. Each user has a separate task file for data storage.

---

## 🔐 Features

- 👤 User Registration
- 🔐 Login System
- ➕ Add Tasks
- 📋 View Tasks
- ✅ Mark Task as Completed
- ❌ Delete Tasks
- 🎯 Priority System (High / Medium / Low)
- 📅 Due Date Support
- 💾 File-based Data Storage

---

## 🧠 Concepts Used

- Structures in C
- File Handling (fopen, fscanf, fprintf)
- Strings Handling
- Functions
- Conditional Statements
- Loops
- Multi-file data system

---

## 🛠️ Technologies Used

- C Language
- GCC Compiler

---

## ▶️ How to Compile

```bash
gcc todo_advanced.c -o todo
```

---

## ▶️ How to Run

```bash
./todo
```

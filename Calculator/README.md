# Simple Calculator in C

A beginner-friendly calculator project developed using the C programming language.

## 📖 About

This project performs basic arithmetic operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

The program takes two numbers and an operator from the user and displays the result.

## 🚀 Features

- Easy to understand
- Beginner-friendly code
- Uses switch-case statements
- Handles division by zero
- Fast and lightweight

## 🛠️ Technologies Used

- C Programming Language
- GCC Compiler

## ▶️ How to Run

### Compile the program

```bash
gcc calculator.c -o calculator
```

### Run the program

```bash
./calculator
```

## 💻 Example Output

```text
===== Simple Calculator =====

Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5

Result = 50.00
```

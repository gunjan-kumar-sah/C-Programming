# 📚 College Academic Management System (C Language)

A complete console-based **College Academic Management System** built using C language. This project manages student records, calculates CGPA, converts CGPA to percentage, finds toppers, and stores data in a file.

It is a **recruiter-level Data Structures project** demonstrating real-world academic management system design.

---

# 🚀 Project Overview

This system helps in managing student academic records in a structured way. It supports:

- Add student records
- Store marks for multiple subjects
- CGPA calculation
- Percentage conversion
- Class average CGPA
- Topper detection
- File storage system

This project simulates a real-world **college ERP academic module**.

---

# ✨ Features

## 👨‍🎓 Student Management
- Add student ID and name
- Store marks for 5 subjects

## 📊 Academic Calculation
- CGPA calculation based on marks
- Percentage conversion from CGPA
- Class average CGPA

## 🏆 Ranking System
- Automatically detects topper student

## 💾 File Handling
- Stores data in `students.txt`

## 📋 Reporting System
- Displays all students in tabular format

---

# 🧠 Concepts Used

- Structures in C (`struct`)
- Arrays (student database)
- Functions (modular programming)
- Loops (data processing)
- File Handling (`fopen`, `fprintf`, `fclose`)
- Pointers (structure passing)
- Basic algorithm design

---

# ⚙️ Working Flow

1. Add student details (ID, Name, Marks)
2. System calculates CGPA and Percentage
3. Store multiple students in memory
4. Display reports and analytics
5. Find topper student
6. Save data into file

---

# 📊 Formulas Used

## CGPA Calculation
```

CGPA = Total Marks / (Subjects × 10)

```

## Percentage Conversion
```

Percentage = CGPA × 9.5

```

# 🚀 Scientific Calculator Using C++

A menu-driven Scientific Calculator developed using C++ and Object-Oriented Programming (OOP) concepts. This project performs both basic and advanced mathematical operations while demonstrating clean and reusable code.

## 📖 About the Project

This calculator allows users to perform various mathematical calculations through a simple menu-driven interface. The project is built using C++ and follows Object-Oriented Programming principles to ensure modularity and code reusability.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Power Calculation
- √ Square Root
- % Modulus Operation
- Menu-Driven Interface
- Error Handling
- OOP-Based Design

## 🧠 Concepts Used

- Conditional Statements

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- GCC / G++ Compiler

## ▶️ How to Compile

```bash
g++ calculator.cpp -o calculator
```

## ▶️ How to Run

```bash
./calculator
```

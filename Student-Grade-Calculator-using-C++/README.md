# 🎓 Student Grade Calculator (C++ OOP)

A C++ program that calculates student percentage and grade using Object-Oriented Programming concepts.

## 📖 About

This project allows users to enter marks of multiple subjects and calculates total marks, percentage, and grade using a class-based approach.

## ✨ Features

- OOP-based design
- Input multiple subject marks
- Calculate total marks
- Calculate percentage
- Grade system (A+, A, B, C, Fail)
- Clean and modular code

## 🧠 Concepts Used

- Classes and Objects
- Encapsulation
- Arrays
- Functions
- Conditional Statements
- Loops

## 🛠️ Technologies Used

- C++
- G++ Compiler

## ▶️ How to Run

```bash
g++ grade_calculator.cpp -o grade
./grade
```
